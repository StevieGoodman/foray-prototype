# Game Template
Roblox game template used for quickly setting up a Rojo workflow suited for game development.

# Configuration
## Toolchain
This project makes use of:
- Rojo (Studio-to-filesystem server)
- Wally (package manager)
- Moonwave (documentation)

## Packages
- Knit
- Observers
- Component
- Option
- Signal
- EnumList
- TableUtil
- WaitFor
- Cmdr
- Promise
- ProfileService
- Roact
